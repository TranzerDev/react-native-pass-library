# react-native-pass-library

A package to open pk pass on iOS

## Installation

```sh
npm install react-native-pass-library
```

## Usage

```js
import PassLibrary from "react-native-pass-library";

// ...

const result = await PassLibrary.multiply(3, 7);
```
